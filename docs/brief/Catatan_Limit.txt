Screening Cicilan Pintar (berdasar trust_score)
TS = 50 → Baseline
TS < 55 → Ditolak
TS ≥ 70 → Layak

Menghitung Trust Score (0–100)
Mulai dari baseline 50, lalu tambah/kurangi poin:
– Umur akun
≥ 30–179 hari → +10
≥ 180 hari → +20

– Riwayat angsuran (maksimal +20 dari tepat waktu)
Tepat waktu → +2 per transaksi (maks +20)
Telat → −5 per transaksi
Gagal bayar → −25

– Frekuensi belanja
≥ 3 transaksi/bulan (3 bulan terakhir) → +5

– Nilai transaksi
Rata-rata > median toko → +5

– Tunggakan aktif
Status DUE/LATE → −10

1 transaksi telat → −15


Menghitung Credit Limit (plafon cicilan)

a. Tentukan limit_base berdasar kebiasaan belanja:
½ transaksi terbesar
½ rata-rata 3 transaksi terbesar
30 % total belanja 6 bulan terakhir
→ Pilih nilai terbesar = Limit Base
(Jika transaksi terbesar > 3× median transaksi, maka tidak dihitung)

b. Tentukan faktor TS:
TS < 55 → 0×
TS 55–59 → 0.5×
TS 60–74 → 1.0×
TS 75–89 → 1.2×
TS ≥ 90 → 1.4×

c. Kalkulasi akhir:
credit_limit = limit_base × faktor_TS (dibulatkan ke ribuan)

Minimum limit: 100.000
Jika ada tunggakan aktif → limit = 0
